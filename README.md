# Simple Quiz

Simple Quiz is a lightweight, responsive web-based multiple-choice quiz application built with plain HTML, CSS, and JavaScript.

## Overview

Simple Quiz displays a sequence of questions with four answer options. When a user selects an option the app highlights the selected answer as correct (green) or incorrect (red), disables further selection for that question, and reveals a `Next` button to proceed. At the end the app shows the user's score and offers a restart.

## Features

- Clean, responsive layout with minimal dependencies
- Interactive selection feedback (correct / incorrect)
- Score display and restart flow
- Easy to extend: questions are stored in `script.js`

## Project Structure

- `index.html` — main HTML file
- `style.css` — visual styles and responsive layout
- `script.js` — quiz logic and UI behavior

## Installation & Usage

1. Clone or copy the project to your local machine.
2. Open `index.html` in a web browser (Chrome, Edge, Firefox).

Optional (using Git):

```bash
git clone <your-repo-url>
cd QuizApp
# open index.html in browser or serve with a static server
```

## Customization

- Edit the `questions` array in `script.js` to add, remove, or change questions and answers.
- Update styling in `style.css` to match your branding.

## Development

No build step is required. Use any code editor (e.g., VS Code) and live-server extensions to preview changes during development.
